## 🚀 快速发布步骤

### 方案 A：Vercel 部署（最快，推荐）

**时间：2-3 分钟**

1. 访问 https://vercel.com
2. 使用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择 "Import Git Repository"
5. 选择 `chinese-pool-game` 仓库
6. 点击 "Deploy"
7. 等待部署完成
8. 获得全球可访问链接！

**示例链接：** `https://chinese-pool-game.vercel.app`

---

### 方案 B：GitHub Pages 部署

**时间：5-10 分钟**

#### 步骤 1：运行部署脚本
```bash
cd chinese-pool-game
chmod +x deploy-to-github.sh
./deploy-to-github.sh
```

#### 步骤 2：按提示输入 GitHub 用户名

#### 步骤 3：在 GitHub 启用 Pages
1. 进入你的仓库页面
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / (root)
5. Save

#### 步骤 4：等待部署
- 等待 1-2 分钟
- 刷新页面查看部署状态
- 获得访问链接

**示例链接：** `https://YOUR_USERNAME.github.io/chinese-pool-game/`

---

### 方案 C：Netlify 拖拽部署（最简单）

**时间：1 分钟**

1. 访问 https://app.netlify.com/drop
2. 将 `chinese-pool-game` 文件夹拖到网页
3. 立即获得访问链接！
