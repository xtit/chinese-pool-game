## 🚀 接下来你需要做的（只需 3 步）

### 第 1 步：在 GitHub 创建仓库

1. 访问 [github.com](https://github.com)
2. 登录你的账号
3. 点击右上角 **+** → **New repository**
4. 填写信息：
   - **Repository name**: `chinese-pool-game`
   - **Description**: `🎱 中式台球游戏 - Chinese Pool Game built with HTML5 Canvas`
   - **Visibility**: 选择 **Public**（公开让全世界看到）
   - ❌ 不要勾选 "Initialize this repository with a README"
5. 点击 **Create repository**

### 第 2 步：推送代码到 GitHub

复制 GitHub 页面上显示的命令，或执行以下命令：

```bash
cd /Users/jarry/.real/users/user-2efca62d7539f55582e4e71179aebf90/workspace/chinese-pool-game

# 关联远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/chinese-pool-game.git

# 推送代码
git branch -M main
git push -u origin main
```

### 第 3 步：启用 GitHub Pages（免费发布）

1. 在你的 GitHub 仓库页面，点击 **Settings**
2. 左侧菜单找到并点击 **Pages**
3. 在 **Build and deployment** 部分：
   - Source: 选择 **Deploy from a branch**
   - Branch: 选择 **main**，文件夹选择 **/(root)**
4. 点击 **Save**
5. 等待 1-2 分钟

🎉 **完成！** 你的游戏将发布到：
```
https://YOUR_USERNAME.github.io/chinese-pool-game/
```

## 🌐 替代方案：使用 Vercel（更快，无需配置）

如果你觉得 GitHub Pages 配置麻烦，可以用 Vercel：

### 方法 A：网页版（最简单）
1. 访问 [vercel.com](https://vercel.com)
2. 用 GitHub 账号登录
3. 点击 **Add New Project**
4. 选择 **Import Git Repository**
5. 找到你的 `chinese-pool-game` 仓库
6. 点击 **Deploy**
7. 完成！获得网址如：`https://chinese-pool-game.vercel.app`

### 方法 B：命令行版
```bash
# 安装 Vercel CLI
npm install -g vercel

# 部署
cd /Users/jarry/.real/users/user-2efca62d7539f55582e4e71179aebf90/workspace/chinese-pool-game
vercel --prod
```
